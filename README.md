## Descripción
Aplicación en React para gestionar películas y series por ver y vistas. Permite filtrar, buscar, ordenar, editar, eliminar y guardar todo en localStorage.


## Cómo correr el proyecto
```bash
git clone 
cd gestor-peliculas-series
npm install
npm start
